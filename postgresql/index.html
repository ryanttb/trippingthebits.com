Serial columns

I ran into an odd issue with serial columns that I thought I should write down. My company is still on PostgreSQL 8.4.3 so I don't know if this is an issue in 9.0.1.

You can create an auto-incremnting ID column in Postgres with the serial keyword:

create table "TestSerial" (
  "TestId" serial primary key,
  "TestVal" text
);
