Windows 8, jQuery Geo, and toStaticHTML

2012-09-10

I attended a Windows 8 DevCamp recently with the intention of installing Microsoft's latest OS for the first time and getting jQuery Geo to work as a native app on top of WinJS.

Did it work?

Yes!

Apart from ajax, jQuery itself works fine on WinJS's Chakra engine. jQuery Geo does not use ajax itself so this is not a problem.

The only change to the jQuery Geo source code I had to make was to surround any HTML text used to dynamically create elements with a Windows 8 JavaScript method: toStaticHtml. This is required when you pass HTML to any jQuery manipulation method such as append.

Before:

.append( '<div />' )

After:

.append( toStaticHTML( '<div />' ) )
