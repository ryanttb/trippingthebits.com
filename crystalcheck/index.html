Crystal Report Checkboxes

2007-12-03

How do you make visual checkbox fields in Crystal Reports? Well, it is certainly not straight forward.

  1. Assuming you have a database already connected to the report, you must create a formula field for each bit column you want to display as a checkbox. Right-click on the Formula Fields section and choose New.
  2. Type in a name, I have been using the naming scheme chkColumnName for each checkbox field.
  3. Click Use Editor.
  4. In the editor that comes up, put the line:
    if {Table.ColumnName} = true then Chr(254) else Chr(168)
    Where Table and ColumnName are appropriate for your database.
  5. Save and close the formula.
  6. Drag the formula from the Formula Fields section onto your report.
  7. This is the most important step: change the font of the text object to Wingdings.

You should now get a checked or unchecked box in your report. I hope this helps someone, it has taken me too long to figure out how to create the appropriate type of field/formula from scratch.
